JavaScript 무엇인가?

JavaScript ( / ˈ dʒ ɑː v ə s k r ɑ t / )는 종종 JS 로 약칭되며 HTML 및 CSS 와 함께 World Wide Web 의 핵심 기술 중 하나인 프로그래밍 언어 입니다 . 2023년 기준으로 웹사이트 의 98.7%는 웹페이지 동작을 위해 클라이언트 측 에서 JavaScript를 사용하며 [10] 종종 타사 라이브러리를 통합합니다 . 모든 주요 웹 브라우저 에는 전용사용자 의 장치 에서 코드를 실행하는 JavaScript 엔진입니다 . JavaScript는 ECMAScript 표준 을 준수하는 높은 수준의 JIT( Just -In-Time) 컴파일 언어입니다 . [11] 동적 타이핑 , 프로토타입 기반 객체 지향 및 일류 기능이 있습니다 . 이는 이벤트 기반 , 기능적 및 명령형 프로그래밍 스타일을 지원하는 다중 패러다임 입니다 . 텍스트, 날짜, 정규식 , 표준 데이터 구조 및 문서 개체 모델 작업을 위한 API( 응용 프로그래밍 인터페이스 ) 가 있습니다 . (돔). ECMAScript 표준에는 네트워킹 , 스토리지 또는 그래픽 기능 과 같은 입출력 (I/O)이 포함되어 있지 않습니다 . 실제로 웹 브라우저나 기타 런타임 시스템은 I/O용 JavaScript API를 제공합니다. JavaScript 엔진은 원래 웹 브라우저에서만 사용되었지만 이제는 일부 서버 및 다양한 애플리케이션 의 핵심 구성 요소입니다 . 이 용도로 가장 널리 사용되는 런타임 시스템 은 Node.js 입니다 . Java 와 JavaScript는 이름, 구문 및 각각의 표준 라이브러리가 유사 하지만 두 언어는 서로 다르며 디자인이 크게 다릅니다.
