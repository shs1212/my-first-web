WEB이란 무엇인가?

WEB 도널드 E. 커 누스(Donald E. Knuth) 가 " 읽을 수 있는 프로그래밍 " 이라고 부르는 것의 첫 번째 구현으로 만든 컴퓨터 프로그래밍 시스템 입니다 . 즉, 설명 텍스트 안에 소스 코드를 삽입함으로써 소프트웨어를 문학 작품으로 만들 수 있다는 아이디어입니다 . 대부분의 프로그래밍 언어 에서 일반적인 관행과 마찬가지로 ) 컴파일러 가 요구하는 순서가 아닌 사람 독자에게 설명하기 편리한 순서로 진행됩니다 . [1] Web은 두 개의 보조 프로그램으로 구성됩니다. 소스 텍스트에서 컴파일 가능한 파스칼 코드를 생성하는 TANGLE과 TeX 를 사용하여 보기 좋게 형식화되고 인쇄 가능한 문서를 생성하는 WEAVE입니다 . CWEB 은 C 프로그래밍 언어 용 Web 버전인 반면 noweb은 Web(이름에 반영됨)에서 영감을 얻었으며 언어에 구애받지 않는 별도의 프로그래밍 도구입니다 . 웹으로 작성된 가장 중요한 프로그램은 TeX 와 Metafont 입니다 . 최신 TeX 배포판은 다른 프로그램인 Web2C를 사용하여 웹 소스를 C로 변환합니다.
