CSS이란 무엇인가?

CSS ( Cascading Style Sheets )는 HTML 또는 XML ( SVG , MathML 또는 XHTML 과 같은 XML 방언 포함 ) 과 같은 마크업 언어 로 작성된 문서의 표시를 설명하는 데 사용되는 스타일 시트 언어 입니다. [1] CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 초석 기술입니다 . [2] CSS는 레이아웃 , 색상 , 글꼴을 포함하여 콘텐츠와 프리젠테이션을 분리 할 수 있도록 설계되었습니다 . [3] 이러한 분리는 콘텐츠 접근성을 향상시킬 수 있습니다 . 프리젠테이션 특성을 지정하는 데 더 많은 유연성과 제어 기능을 제공합니다. 관련 CSS를 별도의 .css 파일에 지정하여 여러 웹 페이지 에서 서식을 공유할 수 있도록 하여 구조적 콘텐츠의 복잡성과 반복을 줄입니다. .css 파일을 캐시하여 파일 과 해당 서식을 공유하는 페이지 간의 페이지 로드 속도를 향상시킵니다. 서식과 콘텐츠를 분리하면 동일한 마크업 페이지를 화면상, 인쇄물, 음성(음성 기반 브라우저 또는 화면 판독기 사용), 점자 기반 등 다양한 렌더링 방법에 따라 다양한 스타일로 표시할 수 있습니다 . 촉각 장치. CSS에는 모바일 장치 에서 콘텐츠에 액세스하는 경우 대체 형식 지정에 대한 규칙도 있습니다 . [4] 계단식이라는 이름은 둘 이상의 규칙이 특정 요소와 일치하는 경우 적용되는 스타일 규칙을 결정하기 위해 지정된 우선 순위 체계에서 비롯됩니다. 이 계단식 우선순위 체계는 예측 가능합니다. CSS 사양은 W3C( World Wide Web Consortium )에서 관리합니다. 인터넷 미디어 유형( MIME 유형 ) text/css은 RFC 2318(1998년 3월)에 의해 CSS와 함께 사용하도록 등록되었습니다. W3C는 CSS 문서에 대한 무료 CSS 유효성 검사 서비스를 운영합니다. [5] HTML 외에도 XHTML , 일반 XML , SVG 및 XUL을 포함한 다른 마크업 언어가 CSS 사용을 지원합니다 . CSS는 GTK 위젯 툴킷 에서도 사용됩니다 .
